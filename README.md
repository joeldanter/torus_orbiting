# Installation
1. Clone the repo
```shell
git clone https://github.com/joeldanter/torus_orbiting.git
```
2. Enter directory
```shell
cd torus_orbiting
```
3. Create virtual environment
```shell
python -m venv .venv
```
4. Activate it
```shell
"./.venv/Scripts/activate"
```
4. Install dependencies
```shell
pip install -r requirements.txt
```

# Usage
```shell
"./.venv/Scripts/activate"
python main.py
```
